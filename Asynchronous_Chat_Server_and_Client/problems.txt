Միաժամանկ  մի քանի client բացելուց էն client-ը որը ուղարկում է message ընկունում է "սպասման ռեժիմ" 
ու չի կարղանում ոչ մի բան ուղարկել քանի դեռ ուրիշ client բան չի ուղարկել որովհետև server-ը բոլորին
ուղարկում  է message բացի իրենից (կարող ենք իրեն դատարկ message հետ ուղարկել), մեկել  client ների մոտ 
Real Time չի արտածվում message-ները, այլ իրենց կողմից ինչ որ message ուղարկելուց հետո
